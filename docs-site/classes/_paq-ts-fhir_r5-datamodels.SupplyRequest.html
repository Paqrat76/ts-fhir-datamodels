    Class SupplyRequest

    SupplyRequest Class

    Remarks

    A record of a non-patient specific request for a medication, substance, device, certain types of biologically derived product, and nutrition product used in the healthcare setting.

    FHIR Specification

    • Short: Request for a medication, substance or device
    • Definition: A record of a request to deliver a medication, substance or device used in the healthcare setting to a particular destination for a particular person or organization.
    • FHIR Version: 5.0.0
